# Socius
I wanted a modern looking, open-source M3 contacts app. I didn't find a good one, so I am making my own.
# Features (finished)
- Read local contacts
- Show display name and profile picture
- See full contacts
- M3 Design
# Features (not finished)
- Sync with CalDAV-Server
- Edit contacts
- See full contacts
# Inspiration
- Google Contacts
